## 1. PT 트레이너의 하루 일과

### 근무 구조
- 대형 센터: 아침 6시 출근 ~ 저녁 8-11시 퇴근, 주 6일 근무
- 소형 스튜디오: 8-10시간이지만 오픈·마감·영업·청소까지 담당
- **분할 근무(Split Shift)**: 오전 피크(06~10시) / 오후 피크(18~23시) 사이 공백 → 하루 12시간 이상 구속

### 수업 횟수
- 소형 센터: 하루 3-4회 (주 10-20회)
- 대형 센터: 하루 8-12회 (월 150-200회 소화 사례도 있음)

### 시간대별 일과

| 시간 | 업무 |
|------|------|
| 06:00-09:00 | 새벽/오전 PT + 운동 영상 촬영 |
| 09:00-11:00 | 개인 운동, 오후 프로그램 구성 |
| 11:00-14:00 | 밀린 운동일지 작성, 식단 피드백, 회원 카톡 답장 |
| 14:00-17:00 | SNS 마케팅, 블로그, 프로그램 설계, 공부 |
| 17:00-22:00 | 피크타임 연속 PT (4-6타임), 신규 회원 응대 |
| 22:00-23:30 | 마감 청소, 매출 정리, 남은 기록 입력, 내일 준비 |

### 4가지 노동 유형
- **육체적 노동** — 코칭, 자세 교정, 시범
- **정신적 노동** — 프로그램 설계
- **감정 노동** — 회원 관리, 동기부여, 영업
- **디지털 노동** — SNS 마케팅, 콘텐츠 제작

## 2. 핵심 고충

### 2-1. 운동 기록의 딜레마

> "운동일지가 중요한 건 아는데, 제대로 쓰려면 시간이 너무 깨지고, 대충 쓰면 쓸모가 없다"

- 세션당 권장 기록 시간: 15분 (수업 전 5분 + 중 5분 + 후 5분)
- 하루 10세션이면 기록에만 **2.5시간** → 현실적으로 불가능
- **실시간 기록 불가**: 수업 중 앱 입력은 코칭 흐름을 끊음
- **중복 노동**: 종이에 메모 → 나중에 앱/엑셀에 옮겨적기
- **비정형 데이터**: "어깨 통증 약간 있음" 같은 정성적 정보를 정형 필드에 넣기 어려움
- **기록 파편화**: 종이/엑셀/카톡/앱에 흩어져 나중에 못 찾음

### 2-2. 회원 관리·소통

- **카톡/DM 과부하**: 업무와 사생활 완전히 혼재
- **소통 채널 파편화**: 카톡, 인스타 DM, 전화, 문자 등 분산
- **후속 관리 압박**: 수업 외 무료 감정노동 (식단 피드백, 안부, 동기부여)
- **개인화 한계**: 20-30명에게 매번 다른 맞춤 프로그램 → 물리적 불가 → 복붙식 프로그램

### 2-3. 구조적 문제
- 저녁·주말 없는 삶
- 불투명한 인센티브 (센터 70-80% : 트레이너 20-30%)
- 센터에 구속되나 수업 없는 시간은 무급에 가까움

### 핵심 악순환

```
시간 없음 → 기록 대충/안 함 → 다음 수업 때 기억 안 남
    → 체계적 관리 불가 → 회원이 성장 못 느낌 → 재등록률 하락
```

## 3. 경쟁 서비스 분석

### 3-1. 한국 시장

#### 센터 CRM (운영자 관점)

| 서비스 | 핵심 기능 | 특징 |
|--------|----------|------|
| **온핏** | 회원권·출입·PT예약·체성분·매출 통합 CRM | 대형 센터에 많이 보급, 센터 제휴 전제 |
| **바디코디** | 예약, QR입장, 매출 실시간, 강사 급여 자동 정산 | 월 ~11.5만원(1년 약정) |
| **스튜디오메이트** | 필라테스/요가 스튜디오 특화 예약·일정 관리 | 합리적 가격, 약정 강요 적음 |
| **다짐매니저** | 키오스크 결제, 입장 통제, 전자계약서 | 기본 월 1.9만원 |
| **어시스트핏** | QR출석, 수업 알림, 직원별 매출 분석 | **전 기능 무료** |
| **체크짐** | 락커·수업횟수 관리 등 필수 기능 집중 | 월 1.4만원대 |
| 핏투비, 피트너스, 트랩핏 등 | 회원권·결제·출입통제·매출통계 | 센터 레벨 운영 초점 |

**공통 한계**: 세트 단위 운동일지, PT 레포트까지 깊게 들어가지 않음

#### PT 기록/회원관리 (트레이너 관점)

| 서비스 | 핵심 기능 | 강점 | 약점 |
|--------|----------|------|------|
| **포인티** | 통증/체형 문진, 퍼스널 레포트 자동 발행, 알림톡 | 회원 앱 설치 불필요 (카톡 발송), 프리랜서도 사용 가능 | UI 개선 요청 존재 |
| **헬린캠프** | 번거로운 인증 없는 회원 추가, 운동일지/일정 자동 공유 | 현재 무료, 빠른 기능 개선 | 도입 센터 수 불명확 |
| **라포** | AI 식단카메라, 스케줄·예약·식단·채팅 통합 | 회원 앱 경험 좋음, 식단 인증 구조화 | B2B 모델 (센터 도입 전제) |
| **트쌤일지** | 엑셀/구글시트 기반 운동일지·회원관리 템플릿 | IT 비친화적 트레이너도 적응, 저렴 | 모바일 UX 제한, 알림 불가 |
| **온핏 매니저** | 앱 내 식단/스케줄/운동 이력 관리 | 개인 SNS 노출 없이 관리 | 센터 솔루션 연동 전제 |
| **플코** | 스포츠 선수 대상 퍼포먼스 측정 리포트 | 전문 피지컬 측정 데이터 | 회당 ~8만원 (비쌈) |
| **센티프-코치** | 트레이너 친화 인터페이스, 라이프 코칭 | 입소문 | 시스템 오류, 예고 없는 점검 |

**현실**: 상당수 트레이너가 여전히 **"센터 CRM + 엑셀/노트/카톡"을 병행**하는 과도기

### 3-2. 해외 시장

| 서비스 | 포지션 | 가격 (50인 기준) | 강점 | 약점 |
|--------|--------|-----------------|------|------|
| **Trainerize** | 온라인/하이브리드 PT 올인원 | 월 ~$175 (23만원) | 프로그램 빌더, 7500+ 운동영상, MyFitnessPal 연동 | 모바일 버그, 비쌈 |

| **PT Distinction** | 고급형 온라인 코칭 | 월 ~$89.90 | 가장 강력한 프로그램 빌더, 가성비 | 학습 곡선 가파름, 영어만 |

**공통 한계**: 한국어 미지원, 한국 PT 문화(회당 결제, 카톡 소통)와 불일치, 월 20만원+ 부담

### 3-3. 앱 비도입 이유

- **"또 다른 일거리"** — 앱이 일을 줄여주는 게 아니라 늘린다는 인식
- **회원 앱 설치 설득의 스트레스** — 결국 종이와 카톡으로 돌아감
- **비용 대비 효용** — 회원 수 적으면 월 구독료가 부담
- **센터 CRM과 중복 입력** — 이중 노동 발생

## 4. 미해결 Gap

### 4-1. 기존 서비스가 못 푸는 것

1. **입력 부담**: "정리해주는 도구"는 많지만, "대신 만들어주는 도구"는 없음
2. **데이터 파편화**: 센터 CRM / 개인 도구 / 카톡 / 회원 앱에 분산
3. **코칭 인사이트 부재**: 어떤 회원이 이탈할지, 어떤 패턴이 효과적인지 분석 없음
4. **감정 노동 경감 불가**: 10명+ 회원에게 각각 다른 톤으로 피드백 → 번아웃

### 4-2. AI 개입 기회

| 영역 | 기존 | AI 적용 후 |
|------|------|-----------|
| 운동 기록 | 직접 타이핑/메모 | 음성·자연어로 말하면 자동 구조화 |
| 회원 리포트 | 수동 작성 또는 없음 | 기록 기반 자동 생성 |
| 프로그램 설계 | 0에서 수작업 | AI 초안 → 트레이너 수정 |
| 회원 메시지 | 일일이 카톡 작성 | AI 초안 → 트레이너 확인 후 발송 |
| 이탈 예측 | 감에 의존 | 데이터 기반 리스크 스코어링 |
| 영상 활용 | 찍고 카톡에 보내고 끝 | 동작 인식, 자세 분석, 변화 추적 |

### 4-3. CoachMate 포지셔닝

```
기존 서비스:  센터 CRM (운영) ←→ PT 기록 앱 (정리)
                                    ↑
CoachMate:              AI 코칭 코파일럿 (생성)
                   "기록·피드백을 대신 만들어주는 레이어"
```

기존 도구를 대체하는 것이 아니라, 그 위에 올라가는 **AI 레이어**.
