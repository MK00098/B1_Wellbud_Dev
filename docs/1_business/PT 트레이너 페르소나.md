# PT 트레이너 페르소나

## 타겟 사용자

헬스 PT 트레이너 (1회 이상 수업을 진행한 회원을 담당)

### 타겟 확장 순서

1. 헬스 PT 트레이너
2. 재활운동/선수트레이너, 체육학과 학생 (운동지도)
3. 필라테스/요가 등 운동전문가
4. 웰니스 코치 전체 (신체건강, 스포츠, 정신건강, 원데이 클래스)

센터 운영자는 직접 타겟이 아님. 트레이너·회원이 먼저 쓰고 바이럴로 확산.

---

## 하루 일과

### 출근 전 (7:00~8:00)
- 카카오톡 확인 — 회원 문의, 스케줄 변경 요청
- 오늘 수업 스케줄 확인 (보통 하루 6~10회 수업)
- 첫 회원 정보 떠올리기 (어디까지 했지?)

### 오전 수업 (8:00~12:00) — 3~4타임

**수업 전 (5~10분)**
- 지난 수업 기록 확인 (운동, 무게, 세트, 특이사항)
- 오늘 프로그램 구성 (컨디션에 따라 조정)
- 회원 도착 → 컨디션 체크, 간단한 대화

**수업 중 (50분)**
- 운동 지도, 자세 교정, 동기부여
- 머릿속으로 세트/무게/횟수 기억
- 회원 반응 관찰 (통증, 피로도, 자세 변화)

**수업 직후 (5~10분)**
- 다음 회원 오기 전에 기록 남기기 → 시간 부족
- 메모장/노트앱에 급하게 적거나, 아예 안 적고 넘어감
- 회원에게 카톡으로 오늘 운동 내용 보내기 → 귀찮음

### 점심 (12:00~13:00)
- 밀린 기록 정리 (오전 3~4명분)
- 회원 카톡 답장 (식단 질문, 운동 질문)
- 오후 회원 프로그램 준비

### 오후 수업 (13:00~18:00) — 4~5타임
- 오전과 동일 반복
- 회원마다 다른 목표/부상/레벨 기억해야 함
- 수업 사이 5~10분에 모든 걸 처리

### 저녁 피크타임 (18:00~21:00) — 2~3타임
- 가장 바쁜 시간, 기록할 여유 전혀 없음
- 연속 수업으로 체력 소모 큼

### 퇴근 후 (21:00~)
- 하루치 밀린 기록 한꺼번에 정리 → 기억 안 남, 대충 적게 됨
- 내일 수업 회원 프로그램 구성
- 회원 카톡 답장 (오늘 운동 피드백, 식단 피드백)
- SNS 마케팅 (인스타 릴스, 후기 등)

## 핵심 문제

| 문제 | 빈도 | 고통도 |
|------|------|--------|
| 운동 기록 남기기 | 매 수업 (하루 8~10회) | ★★★★★ |
| 기록을 회원에게 보내기 (카톡) | 매 수업 | ★★★★ |
| 지난 기록 확인 → 오늘 프로그램 짜기 | 매 수업 | ★★★★ |
| 회원별 히스토리 기억 (부상, 목표, 진도) | 상시 | ★★★★ |
| 회원 카톡 답장 (식단, 운동 질문) | 수시 | ★★★ |
| 인바디 등 데이터 분석/설명 | 월 1~2회 | ★★ |
| SNS 마케팅 | 매일 | ★★ |

## 핵심 악순환

```
시간 없음 → 기록 대충 적거나 안 적음
    → 다음 수업 때 뭐 했는지 기억 안 남
        → 프로그램 체계적 관리 안 됨
            → 회원이 성장 느끼기 어려움
                → 재등록률 하락
```

트레이너 대부분이 기록의 중요성은 알지만, 현실적으로 시간이 없어서 못 하는 상황.
